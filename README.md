## To run the app 
1. First open a terminal and navigate to `/Book Store App - A3` directory
2. run `mongo`
3. inside mongo shell: 
    a. run `use my-library`
4. Then open a new terminal and navigate to `/Book Store App - A3` directory
5. run `node app.js`
